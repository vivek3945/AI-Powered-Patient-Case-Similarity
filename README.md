# AI-Powered Patient Case Similarity

## Overview

AI-powered healthcare solution that identifies similar patient cases using Machine Learning and Natural Language Processing to support clinical decision-making.

## Features

- Patient similarity analysis
- Machine Learning predictions
- Clinical decision support
- Healthcare chatbot
- Interactive dashboard

## Technology Stack

### Frontend
- React.js
- HTML
- CSS

### Backend
- Python
- Flask

### Machine Learning
- Random Forest
- Scikit-Learn
- NLP
